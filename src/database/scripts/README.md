# Scripts de Base de Datos

Este directorio contiene scripts utilitarios para la gestión y generación de informes de la base de datos `MarIADono3DB.sqlite`.

## Contenido

-   `generateWeeklyBotMessageReport.js`: Genera un informe semanal de la cantidad de mensajes enviados por cada bot.
-   `seedConversationsLog.js`: Script para poblar la tabla `ConversationsLog` con datos de ejemplo para pruebas y desarrollo.
-   `generateMonthlyBotMessageReport.js`: Genera un informe mensual de la cantidad de mensajes enviados por cada bot.
